# Abhinav Sandru Portfolio

Personal portfolio website built with plain HTML, CSS, and JavaScript.

## Live Site

GitHub Pages URL:
`https://abhinavsandru.github.io/abhinav-portfolio-site/`

If the site is not live yet, enable GitHub Pages in:
`Settings -> Pages -> Deploy from a branch -> main / (root)`.

## Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)

## Sections

- About
- Experience
- Projects
- Research Seminars & Presentations
- Skills
- Certifications
- Contact

## Run Locally

Open `index.html` directly in a browser, or run a simple local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
