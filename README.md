# Abhinav Sandru — Portfolio Site

A fast, clean, single‑page portfolio built with plain HTML/CSS/JS.

## Quick start (local)
1. Download the folder.
2. Open `index.html` in your browser.
   - Or run a tiny web server:
     - Python: `python -m http.server 8000`
     - Then open: `http://localhost:8000`

## Deploy (GitHub Pages)
1. Create a new repo (e.g., `abhinav-portfolio`).
2. Upload these files to the repo root.
3. In GitHub: **Settings → Pages → Deploy from branch**, select `main` and `/root`.
4. Your site will appear at `https://<username>.github.io/<repo>/`.

## Customize
- Edit text content in `index.html`.
- Styling in `styles.css`.
- Behavior in `script.js`.
